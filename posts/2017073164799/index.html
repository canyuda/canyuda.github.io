数据库-插入、更新、删除数据

2017-07-31
  • 数据库
   预计 1 分钟读完

略

学习目标

  • 掌握如何向表中插入数据
  • 掌握如何更新数据
  • 熟悉如何删除数据

插入数据

为表的所有字段插入数据

INSERT INTO 表名 (字段列表) VALUES (与字段列表对应的数据);

为表指定字段插入数据

同上,不过字段的列表可以书写指定字段,未书写的字段为默认值,如果没有默认约束则为空,如果有不为空约束,则报错.

同时插入多条数据

INSERT INTO 表名 (字段列表)
VALUES (与字段列表对应的数据),(与字段列表对应的数据),(与字段列表对应的数据),.....;

用逗号连接数据.

讲查询结果插入到表中

INSERT INTO tb1 (col_list1)
SELECT (col_list2) FROM tb2;

要求col_list1和col_list2是一一对应的.

更新数据

UPDATE tb_name
SET col_name1 = value1,col_name2 = value2,col_name3 = value3,...
WHERE (条件);

删除数据

DELETE FROM tb_name [WHERE (条件)];

WHERE建议写,要不然数据全部删除,有点爽.
