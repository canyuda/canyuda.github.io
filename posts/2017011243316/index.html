更新了HEXO博客的插件主要有:

  1. hexo-reading-time 根据字数得到预计阅读时间.

具体用法

hexo-reading-time

根据字数得到预计阅读时间

安装插件

npm install --save hexo-reading-time

用法

添加代码到 post.ejs 和 index-item.ejs 中

基础用法:

<%- readingTime(page.content) %>

定制用法:

<span>预计 </span><%- readingTime(page.content,'分钟读完',250) %>

where:
'分钟.' - 数字后的文字,例如会显示:”3 分钟.”,默认显示”3 min.read”
300 - 数字,用于表示每分钟阅读字数,默认为150,正常人阅读速度大概为300字/分钟

hexo-url-image

安装插件

npm install --save hexo-url-image

用法

  1. _config.yml 里面配置 image_server_url: http://XXX/ 作为图库
  2. _post/xxx.md 的配置区域加上 url_image: can.png
  3. themes\XXX\layout\_partial\archive.ejs 或者其他地方加上
     <% if (post.url_image){ %>
   <img src="<%- post.url_image %>">
 <% } %>
  4. 在生成html时, 会读取 _config.yml 里面的 image_server_url 和 文章里面的 url_image, 拼接成为一个<img>标签

hexo-helper-qrcode

安装

npm install --save hexo-helper-qrcode

使用

找到一个合适的位置对应的ejs文件, 使用如下代码

<img src="<%- qrcode(url) %>">

<!-- white margin, default 0 -->
<img src="<%- qrcode(url, {
    margin: 2
}) %>">

<!-- size of one module in pixels, default 6 -->
<img src="<%- qrcode(url, {
    size: 4
}) %>">

END
