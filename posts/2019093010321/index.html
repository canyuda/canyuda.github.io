活动图

基础

  • **XXX** 表示加粗

开始,结束,终止

  • start 开始
  • stop 结束
  • end 终止

条件判断

  • if 条件框
  • then yes判断线上文字
  • else no判断线上文字
  • endif 条件判断结束符

多条件判断

  • elseif 多条件框

末判断循环

  • repeat 循环体框选
  • while 循环判断
  • is 循环箭头文字

首判断循环

  • endwhile 循环体结束标识

并行

  • fork 并行标识
  • fork again 并行标识
  • end fork 结束并行标识

注释

  • floating 无箭头标识
  • note 注释声明
  • [left|right] 位置标识
  • //XXX// 斜体字
  • ==== 分割线
  • * 小圆点
  • ""XXX"" 代码块
  • <b>XXX</b> html符

颜色

  • #{颜色名}:xxx; 指定颜色名
  • #{颜色名A}/{颜色名B}:xxx; 指定渐变颜色, 左上/右下
  • #{颜色名A}\{颜色名B}:xxx; 指定渐变颜色, 左下\右上
  • #{颜色名A}|{颜色名B}:xxx; 指定渐变颜色, 左|右
  • #{HEX}:xxx 指定HEX值(长短HEX均可)

颜色取值

箭头样式

  • -> 后接箭头文字
  • -[#blue]-> 指定颜色
  • -[#green,dashed]-> 指定颜色和样式, dashed 粗粒度虚线, dotted 细粒度虚线, bold 粗实线

分组

  • partition 分组

泳道

  • |泳道名| 泳道名
  • |#{颜色}|泳道名| 指定颜色的泳道

无下文

  • detach 无下文

岔路

  • split 岔路
  • split again 岔路
  • end split 岔路停止

块样式

  • ;, |, <, >, ], /, } 各种样式
